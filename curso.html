Desenvolvimento Web

Ministrador das aulas: Matheus Vinícius

Duração: 12/10/2024 - 02/11/2024 (4 aulas)

Horário: 08:30 - 12:30

Local: Unifametro - Bloco B - Sala 103

Um curso de desenvolvimento web que aborda os fundamentos essenciais de HTML, CSS e JavaScript, capacitando os alunos a criarem sites interativos. Inclui práticas de design responsivo, assegurando que os sites funcionem bem em diversos dispositivos. Além disso, os estudantes são introduzidos ao framework React, aprendendo a desenvolver interfaces de usuário dinâmicas e eficientes.

Aulas:
1 - Fundamentos de HTML e CSS
2 - Introdução ao JavaScript
3 - Design Responsivo
4 - Introdução ao React
